Jak uruchomić (XAMPP):
1) Rozpakuj folder 'car-reviews-autoinstall' do C:\xampp\htdocs\
2) Uruchom Apache i MySQL w XAMPP.
3) Wejdź: http://localhost/car-reviews-autoinstall/
   - API automatycznie utworzy bazę 'car_reviews' i tabelę 'reviews' przy pierwszym wywołaniu.
4) Jeśli masz inne dane DB, edytuj api/config.php i odśwież stronę.
